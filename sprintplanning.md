# 🚀 PortSafe 2.0 — Planejamento Simplificado de Sprints (PI3)

Projeto organizado em **4 sprints mensais**, com desenvolvimento concentrado na **última semana de cada mês**.

---

# 🥇 Sprint 1 — Planejamento

## 🎯 Objetivo
Organizar o projeto antes do desenvolvimento.

## 📌 Atividades da Semana
- Criar **Backlog do Produto**
- Escrever **User Stories**
- Criar **cards no board (Scrum/Kanban)**
- Definir responsáveis por área
- Desenhar visão geral da arquitetura (rascunho)

## ✅ Resultado Esperado
- Backlog organizado  
- Board com cards criados  
- Escopo inicial validado pelos professores  

---

# 🥈 Sprint 2 — Base do Sistema

## 🎯 Objetivo
Criar estrutura inicial funcional.

## 🔧 Backend
- Criar projeto ASP.NET Core
- Configurar banco PostgreSQL
- Criar entidades principais
- Implementar CRUD básico

## 📱 Mobile
- Criar projeto Expo
- Tela de login simples
- Conectar com API

## 🎨 Interface
- Definir identidade visual
- Criar protótipo inicial do dashboard

## ✅ Resultado Esperado
- API funcionando localmente  
- Login funcional  
- Estrutura base pronta  

---

# 🥉 Sprint 3 — Integração IoT

## 🎯 Objetivo
Conectar sistema ao dispositivo IoT via MQTT.

## 🔧 Backend
- Implementar publicação MQTT
- Atualizar status de entrega

## 🤖 IoT
- Configurar broker local
- ESP32 receber comando
- Simular abertura

## 🎨 Dashboard
- Exibir status do locker
- Lista de entregas

## ✅ Resultado Esperado
- Comunicação MQTT funcionando  
- Locker simulando abertura  
- Status atualizado no sistema  

---

# 🏅 Sprint 4 — Integração Final + Cloud

## 🎯 Objetivo
Integrar tudo e subir para nuvem.

## ☁️ Cloud
- Publicar API na AWS
- Configurar banco em nuvem

## 🔄 Integração Final
- Testar fluxo completo:
  - Criar entrega
  - Abrir locker
  - Confirmar retirada

## 📄 Finalização
- Ajustes finais
- Documentação
- Preparação da apresentação

## ✅ Resultado Esperado
- Sistema funcionando ponta a ponta  
- Deploy em nuvem  
- Projeto pronto para apresentação  

---

# 🏁 Resultado Final

- 📱 Aplicativo mobile funcional  
- 🤖 Integração IoT  
- 🔌 Comunicação MQTT  
- 🖥 Dashboard com métricas básicas  
- ☁️ Deploy em nuvem  
- 📄 Documentação organizada  
- 🔄 Aplicação do Scrum
