# 📚 Documentação -- PortSafe 2.0

Este repositório é destinado exclusivamente à **documentação oficial do
projeto PortSafe 2.0**, desenvolvido no contexto do Projeto
Interdisciplinar (PI3).

## 🎯 Finalidade

Centralizar, organizar e versionar toda a documentação técnica e
metodológica do projeto, incluindo:

-   📌 Arquitetura do sistema\
-   📊 Planejamento de sprints\
-   🧠 Modelagem de dados\
-   🔄 Integração vertical e horizontal\
-   📡 Documentação IoT e MQTT\
-   ☁️ Estrutura de nuvem (AWS)\
-   📱 Fluxos do aplicativo mobile\
-   🖥 Interfaces industriais\
-   📄 Artigo científico\
-   📋 Backlog e artefatos Scrum

------------------------------------------------------------------------

## 🗂 Organização da Documentação

A documentação está organizada por áreas do projeto, respeitando a
estrutura interdisciplinar definida no PI3:

-   **IoT**
-   **Backend**
-   **Cloud**
-   **Mobile**
-   **Interface Industrial**
-   **Gestão e Scrum**
-   **Artigo Científico**

------------------------------------------------------------------------

## 🔎 Observação

Este repositório **não contém código-fonte**.\
Os códigos do projeto estão organizados em repositórios separados dentro
da organização.

------------------------------------------------------------------------

## 🔄 Controle de Versão

Toda alteração na documentação deve ser versionada via Pull Request,
garantindo rastreabilidade e histórico técnico das decisões
arquiteturais.
